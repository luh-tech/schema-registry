# luhtech-schema

Schema-first validation, classification, and authoring substrate for the LuhTech Holdings platform.

## Status

**T0 (scaffold).** Read-only validation surface.

## Install (development)

```bash
pip install -e tools/luhtech-schema
```

## Capabilities at T0

- `luhtech-schema validate <path>` — validate a single file.
- `luhtech-schema validate registry` — walk the registry and run canonical compliance checks on every `*.schema.json` file.
